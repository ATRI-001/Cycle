# Cycle

这是一个用于学习游戏开发与 Git 版本控制的练习项目。

## 目标

- 探索游戏开发的基础知识与实现方法
- 练习 Git 工作流程，掌握版本控制技巧
- 记录学习过程中的想法与实验

## 学习内容

### 游戏开发
- 游戏引擎基础
- 游戏逻辑与机制设计
- 资源管理与性能优化

### Git 练习
- 仓库初始化与基本操作
- 分支管理与合并
- 提交规范与日志查看
- 远程仓库协作

## 开始

```bash
git clone https://github.com/ATRI-001/Cycle.git
```
